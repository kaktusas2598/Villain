Detailed Description

Main engine class containing game loop.

Responsible for managing game loop with semi-fixed time step. Initialises most of the engine's subsystems, including but not limited to Window, RenderingEngine, RigidBodyWorld, EventDispatcher and client Application. Also handles input events and allows using Lua config script to setup startup settings.
